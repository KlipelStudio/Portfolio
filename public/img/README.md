# Imagens

Coloque aqui as fotos das miniaturas (ex: `goblin.jpg`).

No `.md` da miniatura, referencie com caminho a partir da raiz pública:

```yaml
imagem: "/img/goblin.jpg"
galeria:
  - "/img/goblin-detalhe.jpg"
```

URLs externas (`https://...`) também funcionam. O site adiciona o prefixo
`/Portfolio` automaticamente nos caminhos locais.
